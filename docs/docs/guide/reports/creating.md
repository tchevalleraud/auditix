# Creating Reports

Auditix generates professional DOCX compliance reports that you can customize and share.

## Creating a Report

1. Navigate to **Reports** in the sidebar
2. Click **New report**
3. Fill in:
   - **Name** — The report name
   - **Description** — Optional description
   - **Language** — Report language (English, French, German, Spanish, Italian, Japanese)
   - **Type** — General (single report) or Node (one report per node)

<!-- ![Create report](../../../static/img/screenshots/report-create.png) -->

## Report Editor

The report editor lets you configure every aspect of your report:

<!-- ![Report editor](../../../static/img/screenshots/report-editor.png) -->

### Metadata

- **Title** and **Subtitle** — Displayed on the cover page
- **Authors** — List of report authors
- **Recipients** — List of report recipients
- **Revisions** — Version history of the report

### Content Blocks

Build your report content using blocks:

- **Heading** — Section titles (H1 to H6)
- **Paragraph** — Rich text content
- **Table** — Data tables
- **Compliance summary** — Auto-generated compliance results
- **Node information** — Auto-filled node details
- **CLI output** — Formatted command output

### Options

- **Table of contents** — Auto-generated from headings
- **Authors page** — List of authors and recipients
- **Revision page** — Document revision history
- **Illustrations page** — List of tables and figures

## Generating the Report

1. Configure your report content
2. Click **Generate**
3. Wait for the generation to complete (runs in background)
4. Download the DOCX file

<!-- ![Report download](../../../static/img/screenshots/report-download.png) -->

### Node Reports

When using the **Node** report type:
- Select which nodes to include
- One DOCX file is generated per node
- Each report is personalized with the node's data
