# Manufacturers

Manufacturers represent the hardware vendors of your network equipment (e.g., Cisco, Juniper, Fortinet, Aruba).

## Creating a Manufacturer

1. Navigate to **Manufacturers** in the sidebar
2. Click **New manufacturer**
3. Fill in the details:
   - **Name** — The manufacturer name (e.g., "Cisco")
   - **Logo** — Upload an optional logo image

<!-- ![Create manufacturer](../../../static/img/screenshots/manufacturer-create.png) -->

## Manufacturer List

The manufacturer list shows all registered vendors with their logo, name, and the number of associated models.

<!-- ![Manufacturer list](../../../static/img/screenshots/manufacturer-list.png) -->

:::tip
Create your manufacturers first, then add models for each one. Models define the connection scripts and collection commands specific to each equipment type.
:::
