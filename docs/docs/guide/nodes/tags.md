# Tags

Tags help you organize and categorize your nodes. Each tag has a name and a color.

## Creating a Tag

1. Navigate to **Tags** in the sidebar
2. Click **New tag**
3. Fill in:
   - **Name** — The tag label (e.g., "Production", "Core", "Branch Office")
   - **Color** — Pick a color for visual identification

<!-- ![Create tag](../../../static/img/screenshots/tag-create.png) -->

## Using Tags

Tags can be assigned to nodes during creation or from the node settings page. They appear as colored badges on the node list and node detail pages.

<!-- ![Tags on nodes](../../../static/img/screenshots/tags-on-nodes.png) -->

Tags are also useful for:
- **Filtering** nodes in the list view
- **Organizing** collections by tagging them
- **Grouping** nodes visually by function, location, or criticality
